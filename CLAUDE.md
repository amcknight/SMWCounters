# SMWCounters — project instructions

## Git workflow

Committing and merging in this repo **without asking first is authorized**,
including directly to `master`. Branch when it helps isolate a larger piece of
work, but a feature branch is not required for small changes. This overrides the
global "commit/merge only when asked, branch first off the default branch"
default — that guard still applies to other repos, just not this one.
